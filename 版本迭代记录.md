# 版本迭代记录

## v2.0.3 (2026-02-11)
- **新增特性**：
  - 飞书自动同步功能：支持本地数据变更后 5 秒自动上传到飞书，无需手动点击备份。
  - 为飞书同步模块添加了静默模式，避免频繁打扰。
- **UI优化**：
  - 在飞书配置面板添加“自动同步”开关。
  - 优化了同步日志显示。
- **Bug修复**：
  - 修复了脚本版本号导致的代码缓存问题（v2.0.3）。

## v2.0.1 (2026-02-11)
- **新增特性**：
  - 飞书 Bitable 集成：支持将学习主题、知识点、练习记录同步到飞书多维表格。
  - 增加 Verce Serverless Function 代理以解决跨域问题。
- **优化**：
  - 改进 Tab 切换逻辑，支持更多功能模块。
  - 优化 Vercel 缓存策略。
